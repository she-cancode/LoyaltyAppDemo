# 🏆 Loyalty Rewards App
A JavaFX application built as part of my **Information Technology** studies. This project demonstrates Object-Oriented Programming (OOP) and custom UI styling. Idea was inspired by a Vibe-coding Hackathon

## ✨ Aesthetic & Theme
The application features a "Cute" UI with a **Pastal** color palette. It uses custom CSS to provide rounded corners and soft pastel highlights.

## 🚀 Features
- **User Management:** Tracks user points and tier levels.
- **Loyalty Engine:** Custom logic to calculate points based on spend ($100 = 10 points$).
- **Rewards Catalog:** A list of redeemable items with varying point costs.
- **Modern UI:** Styled entirely with JavaFX CSS.

## 🛠️ Tech Stack
- **Language:** Java 17
- **Framework:** JavaFX (Modular)
- **Build Tool:** Maven
- **IDE:** VS Code / IntelliJ

## 📂 Project Structure
- `User.java`: Logic for point accumulation and redemption.
- `Reward.java`: Data model for reward items.
- `style.css`: The custom styling.
